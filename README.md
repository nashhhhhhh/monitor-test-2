🛠 Tech Stack
Backend: Python (Flask), Pandas, NumPy, SQLite3

Frontend: HTML5, CSS3, JavaScript (ES6), Chart.js

Integrity: Custom Heartbeat engine with NaN-patching for industrial data.

⚙️ Quick Start
Install Dependencies:

Bash
pip install flask pandas openpyxl numpy
Run Server:

Bash
cd backend
python app.py
Access Dashboard: http://127.0.0.1:5000

🖥 Features
Unified Heartbeat: Pings 8+ subsystems every 30s for real-time health checks.

Fault Tolerance: Robust CSV parsing skips bad lines and handles missing sensor values.

Deep-Link Navigation: Cards link directly to specialized department sub-folders.
